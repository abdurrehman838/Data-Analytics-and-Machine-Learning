## 📊 Core Topics Covered

* **Exploratory Data Analysis (EDA):** Handling missing values, outlier detection, and distribution analysis.
* **Feature Engineering:** Data scaling, normalization, one-hot encoding, and correlation matrices.
* **Supervised Learning (Regression):** Predicting continuous numerical targets using Linear and Lasso techniques.
* **Supervised Learning (Classification):** Evaluating risk and category targets using Logistic Regression and Decision Trees.
* **Model Evaluation Metrics:** Accuracy, $R^2$ Score, Mean Absolute Error (MAE), and Confusion Matrices.
